blend

This conceptual design project takes an arbitrary formal operation, “blend,” to create a form using randomly shaped blocks.Through a series of incremental steps, doors, stairs, columns etc. were added to the form to finally create a “building”. This project explores the transition from abstracted forms/concepts to a more realistic rendition of a building. “Blend” operates in the mixing of parts, which are merged together to create a single entity on the exterior. On one hand, interior walls articulate the original parts. On the other hand, interior walls articulate the awkwardness created through intersection betwen parts as a result of blending.
